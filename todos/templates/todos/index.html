What is it all about?

This is simple todo-list application, which allows you to create and manage your tasks.

How-to

To add one, type query string as follows:

And press Enter.

What every element in query string means?

  • Buy 2.5% milk — title for task
  • ^tomorrow — deadline. Any date value
  • #household #unimportant — list of tags for task
  • !2 — priority. Random number from 1 to 100. Allows you to sort your tasks by priority as you wish

It's pretty simple and intuitive, try it yourself.

Interface guide

To edit existing task, double click on it. Enter to save.
To remove: click on -icon at right corner of table row with a task.
And to mark it as complete: click icon.
Button Clear completed just under the table deletes a tasks, which marked as completed.
